# Jan Bos Voorraad V3

Snelle magazijn-app met:

- Jan Bos-logo
- productfoto's
- grote knoppen -1, -5, +1 en +10
- QR-code per product
- automatische lijst met bij te bestellen producten
- Excel-export
- behoud van de bestaande database

## Uploaden

Upload alle bestanden uit deze map naar de bestaande GitHub-repository `jan-bos-voorraad` en vervang de bestaande bestanden. Render start daarna automatisch een nieuwe deploy.

De Dockerfile in deze versie is gecontroleerd en bevat alleen geldige Docker-instructies.
